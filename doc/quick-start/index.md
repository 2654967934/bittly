# 快速开始

Bittly 是一个支持多种通讯方式和通讯协议的通讯调试工具。 支持串口，网络，蓝牙，HTTP，Websocket，MQTT以及Modbus；支持通过指令绑定快速实现上位机控制面板；支持自动化功能测试以及单元测试；支持通过插件的方式扩展 Bittly 功能。
